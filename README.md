# vanilla-js-todo-app
A simple vanilla JavaScript to-do/wishlist app. Add items, check them off, delete them — your list is saved to localStorage so it persists across refreshes.

Features


Add, complete, and delete items
Persistent storage via localStorage
Built with vanilla HTML, CSS, and JavaScript — no frameworks


Key concepts used


Event delegation for handling clicks on dynamically created elements
Immutable state updates (.map() / .filter()) instead of direct mutation
Single source of truth: the array drives the UI, not the other way around


Running it

Just open index.html in a browser.
